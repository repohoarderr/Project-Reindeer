# Project-Reindeer

# Prerequisites to running the code;
1. Download Node.js from https://nodejs.org/en/download/prebuilt-installer
2. Walk through the installation, accept the defaults. 

# Running (In VSCode):
1. Copy the repo to your local machine. [Here](https://code.visualstudio.com/docs/sourcecontrol/overview) is a guide on using Git in VSCode. 
2. **Create a .gitignore file and put "node_modules/" in it IF it doesn't already exist. Make sure the file is under the Front-End/ folder.**
3. Open the terminal with Ctrl+` and move to the Front-End/ directory if you are not there already.
4. Run "npm i" to download all the necessary node packages. 
5. Split the terminal screen to contain two terminals (Ctrl+Shift+5)
6. Run "npm run client:dev" in one terminal to compile the application. 
7. Run "npm run server:dev" in the second terminal to start the server.
